# Canva Infographic Builder

## Overview

Step-by-step system for building professional, tech-themed infographics in Canva aligned with Cyber Sloth Empire's visual identity.

## Design Tokens (Canva)

### Colors
```
Background:     #0A0F1C or #0D1117
Surface/Cards:  #1A1F35 or #161B22
Accent 1:       #00D4AA (Cyber Teal)
Accent 2:       #7B61FF (Electric Purple)
Accent 3:       #FF6B35 (Alert Orange)
Text Primary:   #F0F0F5
Text Secondary: #8A8FA3
Border/Lines:   #2A2F45
```

### Typography (Canva Fonts)
- **Headlines**: Montserrat Bold or Poppins Bold
- **Subheadings**: Montserrat SemiBold
- **Body**: Open Sans or Inter
- **Data/Code**: Source Code Pro or Roboto Mono

### Sizing
- Title: 36-48pt
- Section Headers: 24-28pt
- Body Text: 14-18pt
- Captions/Labels: 10-12pt

## Infographic Templates

### 1. Process Flow (Vertical)
- Title bar at top
- 5-7 numbered steps flowing downward
- Icons for each step
- Connector lines between steps
- CTA/branding at bottom

### 2. Comparison (Side-by-Side)
- Title bar at top
- Two columns with headers
- Matching rows for each comparison point
- Color-coded (green = good, red = bad)
- Summary row at bottom

### 3. Stats Dashboard
- Title bar at top
- 4-6 stat cards in grid
- Each card: big number + label + icon
- Trend arrows or progress bars
- Source attribution at bottom

### 4. Checklist / Framework
- Title bar at top
- Grouped sections with headers
- Checkbox items or numbered lists
- Color-coded priority levels
- Download CTA at bottom

### 5. LinkedIn Carousel (Multi-Slide)
- Slide 1: Hook/Title (bold, minimal)
- Slides 2-8: One concept per slide
- Final Slide: CTA + handle/branding
- Consistent header/footer across slides

## Build Process

### Step 1: Canvas Setup
- Size: Custom (1080x1920 for social, 1080x1350 for LinkedIn, 800x2000 for tall infographic)
- Background: Set to dark color (#0A0F1C)

### Step 2: Layout Grid
- Add guide lines for margins (40px all sides)
- Create sections with colored rectangles (Surface color, rounded corners 12px)
- Space sections 20-24px apart

### Step 3: Content Hierarchy
- Title section: Full width, centered, accent color underline
- Section headers: Left-aligned, accent color text or pill background
- Body content: Left-aligned, secondary text color
- Data points: Centered in cards, large bold numbers

### Step 4: Icons & Graphics
- Use Canva's icon library — search for tech/security terms
- Color icons with accent colors
- Keep consistent icon style (outline OR filled, not mixed)
- Size: 40-60px for section icons, 24-32px for inline

### Step 5: Effects & Polish
- Card shadows: subtle drop shadow (blur 10, opacity 20%)
- Accent glows: on key elements (blur 20, accent color, opacity 30%)
- Gradient overlays: subtle, accent color to transparent
- Border: 1px border on cards using Border color

### Step 6: Branding
- Bottom bar: Cyber Sloth Empire logo + website + handle
- Consistent footer across multi-slide carousels
- QR code for lead generation if applicable

## Rules

- Always use dark background — never white/light
- Maximum 3 accent colors per infographic
- Text must be readable at mobile size
- Every infographic needs a clear visual hierarchy
- Include source citations for any statistics
- Brand footer on every piece
- Export at 2x resolution for clarity
- Test readability by viewing at 50% zoom
