# Brand Refresh Pipeline

You are running the Maycrest Group's brand evolution playbook. Not a rebrand — a refresh. We're sharpening the blade, not melting it down. The user has identified a brand that needs updating.

## Stage 1: Brand Audit

Invoke the `maycrest-create:brand-guardian` skill. Assess the current state:
- Visual identity inventory (logo, colors, typography, imagery style)
- Brand voice and tone analysis (how does it sound right now?)
- Consistency audit across touchpoints (website, social, product, collateral)
- Competitive positioning (how does it sit relative to competitors?)
- Audience perception gap (how do they want to be seen vs. how they're seen?)

Deliver a **Brand Health Score** (1-10) with specific findings for each dimension.

## Stage 2: Visual Evolution

Invoke the `maycrest-create:ui-designer` skill. Propose the visual refresh:
- Color palette evolution (subtle shift or full rework, with rationale)
- Typography updates (hierarchy, readability, personality)
- Component/UI pattern modernization
- Iconography and illustration style direction
- Photography/imagery guidelines update
- Before/after comparisons for key brand elements

Present 2-3 visual direction options ranging from conservative refresh to bold evolution.

## Stage 3: Narrative Direction

Invoke the `maycrest-create:visual-storyteller` skill. Craft the brand story:
- Updated brand narrative (who are we now, who are we becoming)
- Voice and tone guidelines refresh
- Key messaging framework (tagline, elevator pitch, value props)
- Visual storytelling direction (what does the brand look like in motion?)
- Mood board and reference collection

## Output

Deliver the Brand Refresh Package:

### Brand Health Report
- Current state assessment and scores
- Key opportunities identified
- Recommended evolution level (tune-up / refresh / overhaul)

### Visual Direction
- Recommended option with full rationale
- Updated style guide elements
- Migration plan (what changes first, what can wait)

### Narrative Framework
- Updated messaging hierarchy
- Voice and tone guide
- Storytelling templates for key channels

### Implementation Roadmap
- Priority order for applying changes
- Quick wins (can do this week)
- Medium-term updates (next 30 days)
- Full rollout plan (90 days)

A brand refresh done right feels like the brand finally became who it was always trying to be.
