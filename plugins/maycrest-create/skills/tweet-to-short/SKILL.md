# Tweet-to-Short Converter

## Overview

Convert written posts (tweets, LinkedIn posts, threads) into camera-ready TikTok, Reels, and YouTube Shorts scripts for Corey Maypray.

## Script Structure

### Standard Format (30-60 seconds)

```
HOOK (0-3s): [What you say first — must stop the scroll]
SETUP (3-10s): [Frame the problem or context]
BODY (10-40s): [Deliver the insight in 2-3 beats]
PAYOFF (40-55s): [The "aha" moment or call to action]
CLOSE (55-60s): [Tag, CTA, or signature sign-off]
```

### Quick Hit Format (15-30 seconds)

```
HOOK (0-3s): [Punchy opener]
POINT (3-20s): [One clear insight, fast]
CTA (20-30s): [Follow, comment, share]
```

## Delivery Cues

Use these markers in scripts:
- `[PAUSE]` — dramatic beat, let it land
- `[LEAN IN]` — get closer to camera, lower voice
- `[EMPHASIS]` — slow down, stress this word/phrase
- `[CUT]` — jump cut / scene change
- `[TEXT ON SCREEN: "..."]` — overlay text
- `[B-ROLL: description]` — cutaway footage suggestion

## Voice Matching

- **Cybersecurity content**: Confident expert, slightly intense, "let me put you on game"
- **Consulting/brand**: Polished but real, "here's what I tell my clients"
- **Culture/personal**: Relaxed, funny, "real talk though..."
- **Footwork/music**: High energy, community-coded, authentic

## Rules

- Every script must have a hook that works WITHOUT context
- Write for spoken delivery — contractions, natural rhythm, no "written" language
- Include timing markers for each section
- Add caption text suggestions (different from spoken words where useful)
- Suggest 3 hashtag sets: niche, broad, trending
- If converting from a thread/long post, pick the ONE strongest idea — don't try to cram everything
